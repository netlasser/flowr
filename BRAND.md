# FLOWR — Brand Guidelines

## Brand Name
**FLOWR** (not an acronym). Single word, all caps in the logotype.

## Tagline
- **Primary**: "Where dreams rise through the silence"
- **Secondary** (product context): "Context-first productivity"

## Color Palette
| Token | Value | Usage |
|-------|-------|-------|
| brand-500 | `#10b981` (emerald) | Primary accent, CTAs, active states |
| brand-400 | `#34d399` (emerald light) | Hover states, secondary accents |
| dark-950 | `#020617` | Page backgrounds |
| dark-900 | `#0f172a` | Card/panel backgrounds |
| foreground | `#ffffff` | Body text |
| muted-foreground | `#a1a1aa` | Secondary text |
| border | `#2e2e2e` | Borders |
| whiplash-500 | `#ef4444` | Errors, cognitive switch warnings |
| buffer-500 | `#f59e0b` | Buffer/transition states |

## Typography
- **Display**: Instrument Serif (headlines, logo wordmark)
- **Body**: Inter (UI text, task content, labels)

## Logo
- Wordmark only ("FLOWR" in Instrument Serif)
- No separate icon/symbol mark
- No ® or ™ marks

## Voice & Tone
- Quiet, intentional, anti-hustle-culture
- Speaks to deep thinkers, creators, and quiet rebels
- Precise language, no noise
